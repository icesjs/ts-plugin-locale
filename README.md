# @ices/ts-plugin-locale

## TypeScript's language service plugin that adds IntelliSense for locale components

## Usage

```json5
// tsconfig.json
{
  compilerOptions: {
    plugins: [{ name: '@ices/ts-plugin-locale' }],
  },
}
```
